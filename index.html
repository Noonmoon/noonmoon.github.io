Best Of 3

My role

In the development of this project we split our work into front-end and back-end focused roles. My partner was responsible for user authentication, database and middleware and I was tasked with any front-end related development and programming including html & css, game development, web socket, design, and UI/UX. The user interface I developed ensured a user would never need more than 2 clicks to navigate where they desired.

Challenges I faced

While developing this project one major problem I ran into was client state synchronization. Creating the socket network that would sync clients during play was a challenge I will never forget. Because I was creating my network with sockets this was an even greater challenge because there was no example of this in another P5 application and my mentors had no prior knowledge of client networking. Therefore it was recommended that I did not attempt this feature and stick with local play. However, this was the basis of our project so I had to decided to keep trying until I succeeded just 3 hours before our last day of mentor help. This moment was one that I will remember forever as the day I truly fell in love with the challenge development brings.

Features

* Online multiplayer
* W/L Tracking
* Local Multiplayer
* Account leveling
* 3 Throwback Games

Mocking Bird

In Summary

Mocking Bird was my first personal project I decided to take on after bootcamp. Previous to this project I was intimidated by backend development and had never taken on a role as such. Because of this I wanted an application that would require a lot of backend power and thats what drove me to create a social network from scratch.

Challenges I faced

Although challenging, I was able to create a working backend for my content using MySQL for the first time. I learned important details about the interactions between front-end and back-end elements as well as the full process of creating a backend application from scratch.

Features

* Account Messaging(in development)
* Follow other accounts post stream
* Post content to personal feed

3rd Perspective Design

In Summary

3rd Perspective Design was a family members business site I had decided to fully redesign and build in Wordpress. This was my first experience with content management systems and front-end design for a client. In the end they were very pleased with the site as it was a huge improvement from the last site and I was able to learn a great deal about the design process and using a CMS with clients.

Challenges I faced

With no previous experience using PHP or Wordpress, I was able to fully complete this site with custom post types and 5+ plugins all within 2 weeks of learning and 1 week of development. I believe this showcases the ability I have to absorb new information in foreign environments. I accredit this skill in part to the extensive education I completed under Lighthouse Labs

Features

* Instagram Feed
* Portfolio section with custom post type
* Contact form and google api
* SEO
* Responsive design
